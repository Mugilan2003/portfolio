GUI-Based CRUD Application

A GUI-Based CRUD (Create, Read, Update, Delete) application built using Java and MySQL. This application allows users to manage data efficiently through a user-friendly graphical interface. It demonstrates essential database operations and is designed for beginner-friendly database interaction.

    Key Features:

  • Create: Add new records to the database.
  • Read: Display stored data in a structured format.
  • Update: Modify existing records.
  • Delete: Remove records permanently.
  • User-Friendly UI: Built with Swing for an interactive experience.
  • MySQL Integration: Efficient data storage and retrieval using JDBC.

    Tech Stack:

  • Backend: Java (JDBC)
  • Database: MySQL

This project serves as a foundation for learning database-driven applications and understanding CRUD operations in Java.
